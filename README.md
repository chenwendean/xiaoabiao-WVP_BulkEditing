# 🎛️ WVP 国标平台 - 通道管理工具

> 基于 WVP 开放 API 的通道批量管理桌面工具  
> 支持批量修改通道名称、行政区域，支持 Excel 导入导出

---

## 📸 界面功能

| 界面 | 说明 |
|------|------|
| **登录窗口** | 输入服务器地址、用户名、密码，支持记住密码自动登录 |
| **主界面** | 左侧设备列表 + 顶栏工具栏 + 底部状态栏 |
| **通道列表弹窗** | 双击设备弹出，查看/编辑/导出/导入通道 |
| **进度条** | 查询/导入/导出时实时显示进度 |

---

## 🚀 功能列表

| 功能 | 说明 |
|------|------|
| 🔐 **登录** | 用户名密码登录，自动获取 token，可记住密码 |
| 📋 **设备列表** | 登录后自动查询全部国标设备，复选框多选 |
| 📂 **通道查看** | 双击设备弹出窗口查看所有通道 |
| ✏️ **在线编辑** | 双击通道表格单元格直接修改名称、区域等 |
| ✅ **批量勾选** | 表头 ☐ 一键全选/取消全选 |
| 📝 **批量改区域** | 勾选通道后一键批量修改行政区域编码 |
| 📤 **导出 Excel** | 导出选中设备通道，文件名自动使用设备名称 |
| 📥 **导入 Excel** | 从 Excel 批量导入修改名称、区域、经纬度、厂家 |
| 🔍 **全局通道合并** | 自动查询全局通道 API，补充厂家/经纬度等信息 |
| ⚡ **并发加速** | 可自定义并发线程数（1-32），批量操作飞起 |
| 📊 **进度显示** | 所有耗时操作均有实时进度条 |

---

## ⚙️ 并发设置

默认 **8 路并发**，点击顶栏 `[并发: N]` 按钮可自定义（1-32 路）：

| 并发数 | 适用场景 |
|--------|---------|
| 1 | 调试、服务端限流时 |
| 8 | 默认值，稳定可靠 |
| 16 | 高性能服务器 |
| 32 | 极限性能（需服务器支持） |

并发影响的操作：
- ✅ 分页查询通道列表（按页并发）
- ✅ 合并全局通道数据（按通道并发）
- ✅ 批量修改区域编码
- ✅ 批量导入 Excel

## 🏃 快速开始

### 本地运行

```bash
pip install -r requirements.txt
python main.py
```

### 打包 exe

```bash
pip install pyinstaller
pyinstaller -F -i XYY.ico -w -n "WVP通道管理工具" main.py
```

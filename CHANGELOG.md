## v7.1 — 并发加速 + 进度显示

> 全面优化性能，引入并发控制和进度反馈

### ✨ 新增功能
- **并发设置** — 顶栏右侧新增 `[并发: N]` 按钮，支持自定义 1-32 路并发
- **并发分页查询** — 设备通道查询改为按页并发请求，大幅缩短等待
- **并发批量修改** — 批量改区域、导入 Excel 等操作均使用线程池并发发送 API
- **并发通道合并** — 查询通道后合并全局数据（厂家/经纬度）改为并发处理
- **进度条弹窗** — 双击设备查通道、导出/导入 Excel 均有实时进度条
- **导出默认文件名** — 导出时自动使用设备名称作为默认文件名

### ⚡ 性能优化
- 批量修改：串行 → 8-32 路并发（按设置），加速比 ≈ 并发数
- 通道查询：逐页 → 全页并发，加速比 ≈ 并发数
- 通道合并：逐条查询 → 并发查询全局通道 API
- 列表加载：大量通道分批插入，避免界面卡顿

### 🐛 问题修复
- 关闭右侧空白面板（通道列表占位）
- 移除所有 debug 日志输出（`debug.log`、控制台打印）
- 修复桌面版运行时缺少 requests 模块的问题
- 修复并发查询进度条与页面显示不同步的问题
